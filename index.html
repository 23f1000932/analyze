Data Processing Pipeline

Automated Data Transformation with Python & GitHub Actions

Project Overview

This project demonstrates a robust data processing workflow. It converts an Excel spreadsheet (data.xlsx) into a CSV format (data.csv), processes the data using a Python script (execute.py), and generates an aggregated JSON output (result.json).

The entire process is orchestrated by a GitHub Actions CI/CD pipeline, ensuring code quality, automated execution, and publishing of the final result.json to GitHub Pages.

How it Works

1. Data Source

Starts with raw data in data.xlsx, converted to data.csv for processing.

2. Processing Logic

The execute.py script performs data cleaning, aggregation, and transformation.

3. Output & Deployment

Results are saved to result.json and published via GitHub Pages CI/CD.
